# 🌾 Grama-Angana App

## 📌 Overview
Grama-Angana is a smart village management application designed to connect villagers with local government services digitally. The app helps users access important announcements, complaint registration, village development updates, welfare schemes, and emergency contacts in one platform.

This application aims to improve communication between village citizens and local authorities while promoting digital village development.

---

## 🚀 Features

### 👤 User Management
- User Registration and Login
- Secure Authentication
- Profile Management

### 📢 Village Announcements
- Latest Panchayat announcements
- Event notifications
- Government updates

### 📝 Complaint Management
- Register village-related complaints
- Track complaint status
- Feedback system

### 🏥 Emergency Services
- Emergency contact numbers
- Nearby hospitals and police station information

### 🏛 Government Schemes
- Information about welfare schemes
- Eligibility details
- Application guidance

### 📅 Event Management
- Village event schedules
- Meeting notifications
- Festival information

### 📍 Development Updates
- Village development project tracking
- Work progress updates
